Fridge API

Routes

Note: for all examples, assume I am modifying "bob"'s list and adding/removing "watermelon".

  1. /users/: view all user data
    1. No parameters
  2. /get: view data for one user
    1. Usage: /get?name=bob
  3. /add: add an item to an user's list
    1. Usage: /add?name=bob&item=watermelon
  4. /remove: remove an item from an user's list
    1. Usage: /remove?name=bob&item=watermelon,
